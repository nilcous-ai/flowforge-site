# FlowForge AI (Static Site)

This is a simple, no-code-friendly static website:
- index.html
- styles.css
- script.js

## Publish for free (GitHub Pages)
1) Create a free GitHub account (if you don’t have one)
2) Create a new repository named: flowforge-site
3) Upload these 3 files to the repository root:
   - index.html
   - styles.css
   - script.js
4) In the repo: Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save
5) Wait a minute. Your site will show a link like:
   https://YOURUSERNAME.github.io/flowforge-site/

## Customize
Open index.html and change:
- hello@flowforge.ai (email)
- Location text
- Company name/tagline

That’s it.
